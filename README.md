# Rosco de Habilidades Sociales y Lenguaje

Recurso interactivo educativo para trabajar **comunicación, emociones, convivencia y situaciones cotidianas** en el aula.

## 🎯 Características

- **27 preguntas** en formato de rosco (rueda) alfabético
- **Temática de habilidades sociales**: empatía, respeto, comunicación, resolución de conflictos
- **Interactivo**: marca respuestas como correctas, incorrectas o pasa la pregunta
- **Responsive**: funciona en móviles, tablets y ordenadores
- **Juego visual**: código de colores para estados de respuestas
- **Preguntas para reflexión** al final para profundizar en el aprendizaje

## 📋 Contenidos

El rosco cubre vocabulario sobre:
- **Relaciones sociales**: Amigo, Compañero, Grupo, Respeto...
- **Emociones**: Frustración, Malestar...
- **Comunicación**: Hablar, Escuchar, Interrumpir, Sinceridad...
- **Convivencia**: Normas, Paciencia, Turno, Unirse...
- **Conflictos**: Broma, Disculpa, Queja, Violencia...
- **Entorno digital**: TikTok, WhatsApp, Excluir...

## 🚀 Cómo usar

1. **Abre el archivo** `index.html` en tu navegador
2. **Lee la pista** para cada letra
3. **Escribe tu respuesta** en el campo de texto
4. **Usa los botones**:
   - 📖 **Ver respuesta**: muestra la respuesta orientativa
   - ⏭️ **Pasapalabra**: omite la pregunta por ahora
   - ✅ **Correcta**: marca como acertada
   - ❌ **Incorrecta**: marca como fallida
   - ▶️ **Siguiente**: avanza a la próxima pregunta pendiente
   - 🔄 **Reiniciar**: vuelve a empezar desde cero

## 📊 Scoring

El marcador muestra:
- **Aciertos**: respuestas marcadas como correctas
- **Errores**: respuestas marcadas como incorrectas
- **Pendientes**: preguntas sin responder o pasadas

## 🎓 Uso educativo

### Para el alumnado:
- Aprender vocabulario sobre habilidades sociales
- Reflexionar sobre relaciones interpersonales
- Juego interactivo y motivador

### Para docentes:
- Introducción a temas de convivencia
- Base para debates y reflexión grupal
- Actividad para trabajar inteligencia emocional

### Preguntas de reflexión al final:
- ¿Qué palabra te parece más importante para llevarse bien con los demás?
- ¿Qué diferencia hay entre una broma y hacer daño?
- ¿Qué puedes hacer si ves que un compañero se queda solo?
- ¿Cómo puedes pedir ayuda si algo te molesta?
